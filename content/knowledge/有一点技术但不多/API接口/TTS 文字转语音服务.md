# 文字转语音服务TTS

可以用于读小说读网页等

# 开通服务

申请开通Microsoft TTS服务

[Azure 创建资源](https://tts.yfi.moe/help/reg)

记住region和token

# 用法

## 阅读APP

将TTS配置导入到阅读APP中，如 [开源【阅读】看网络小说](siyuan://blocks/20251105174013-8pt8ihp)

* 在手机浏览器中打开网页 [TTS Importer](https://tts.yfi.moe/azure)
* 填入region和Token配置好可直接跳转
